# systems-dsa
This is an implementation of some of the foundational data structures and algorithms of the C++ STL.

## Build & Test (Linux/macOS/WSL)

```bash
./scripts/setup.sh          # configure, build, test (libcxx preset)
./scripts/setup.sh asan     # ASan+UBSan preset
./scripts/ccdb.sh           # generate compile_commands.json for IDEs
```
